gnumake_sample
==============

Example Project for testing GNU Makefile Rule

by cherojeong@gmail.com


You should run script like as belows
#./local_branch.sh

and then type this command

#git branch

  comments
  default_names
  echoing
  errors
  hello
  master
* sample_project

if you can see like it on output, you can change branch locally.
